## 커밋 컨벤션
1. 커밋 메시지는 `커밋 메시지 템플릿` 형식에 맞추어 작성한다.
2. 커밋 메시지는 `한글` 로 작성한다.
3. `커밋 메시지 템플릿`을 사용하는 방법은 `git commit -m "~~~"` 가 아닌 `git commit`만을 입력하면 된다.
4. `커밋 메시지 템플릿`은 제목-본문-꼬리말로 구성되어있다.
5. 제목과 본문은 반드시 작성해야 하고, 꼬리말은 관련된 이슈가 존재하면 작성한다.
